# Coffee Study Planner ☕

A small productivity web app for studying.
It brings together a planner, a Pomodoro timer, notes, and a resource library in one place with a cafe-themed interface.

I made this to have a simple space where I could organize tasks, focus on work sessions, and keep study material without switching between multiple apps.

## Features

* Study planner for creating and tracking tasks
* Pomodoro timer for focus sessions and breaks
* Notes section for lecture notes or quick ideas
* Resource library to save links, PDFs, videos, etc.
* Cafe themed UI to make the workspace feel relaxed

## Tech Stack

HTML,
CSS,
JavaScript

## How to Run

1. Clone the repository

   git clone https://github.com/Sanchi197/coffee-study-planner.git

2. Open the project folder.

3. Open `index.html` in your browser.
